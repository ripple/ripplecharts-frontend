Live Ripple Network

Each point on the map represents a rippled server connected to the network. Hovering over a point will show more information about the server. The color of each point indicates which ledger the server currently matches. These colors correspond to the color of the ledger numbers scrolling underneath the map. The distance between each ledgers is proportional to the close time of the specific ledger.

The lines linking the points on the map represent consensus (i.e., agreement between two servers on the current ledger).

Each circle within the scrolling ledgers represents a new transaction. The circle size corresponds to the transaction size. The types of transactions shown include:

  • Payment: A payment is a transaction in which value is sent from one ripple account to another.
  • OfferCreate: An offer to buy or sell one currency for another is created. Effectively a limit order.
  • OfferCancel: An offer to buy or sell one currency for another is cancelled.
  • TrustSet: A trust line from one account to another is created or deleted.
  • Other: This is includes all other modifications to the ledger.
