Manage Gateways

Select a currency and manage the gateways associated for the currency drop down for the ‘Live Chart’ page. A list of IRBA registered gateways are listed along with a list of other gateways supported by ripplecharts.com for each currency (to find out more about IRBA, click on the link below). Custom gateways that are not listed by default can be added with the “Add gateways manually” field below.

Customizations such as gateway currency pairs and custom gateways are saved in the browser's local storage. Clearing the cache or accessing ripplecharts.com on a different browser or device will restore ripplecharts.com to the default settings.

Select curency to manage gateways for that curency.
